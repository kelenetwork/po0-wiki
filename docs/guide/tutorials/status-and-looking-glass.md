# 如何使用服务状态与 Looking Glass 判断效果

## 用途

帮助用户通过服务状态页和 Looking Glass 了解当前可用性、区域表现和访问路径结果，辅助判断问题是否与本地环境、目标站点或服务状态有关。

## 适合人群/场景

- 想确认 Po0 当前服务是否正常的用户。
- 发现访问变慢、波动或不可达，需要先做基础判断的用户。
- 需要向客服或技术支持提供更完整反馈信息的客户。
