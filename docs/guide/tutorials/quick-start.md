# 快速开始

## 用途

用最短路径帮助新用户完成从确认需求到开始使用的基础流程。

## 适合人群/场景

- 第一次使用 Po0，希望先跑通基础流程的用户。
- 已经明确入口地区和出口/落地，只需要按步骤开始配置的用户。
- 需要快速验证服务是否符合预期的用户。

## 文章大纲

1. 购买前确认使用目标
2. 选择入口地区
3. 选择出口/落地
4. 完成购买与信息确认
5. 按配置说明接入服务
6. 使用状态页和 Looking Glass 做初步验证
