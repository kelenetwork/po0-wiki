# 如何选择出口/落地

## 用途

帮助用户理解 CNIX、Po0 国际出口、RFC 香港、日本等出口/落地选项的差异，并按访问目标选择合适方案。

## 适合人群/场景

- 已经知道自己主要访问目标，但不确定出口/落地怎么选的用户。
- 需要在香港、日本、国际出口等不同方向之间做取舍的客户。
- 对访问速度、稳定性、目标地区可达性有明确要求的用户。

## 文章大纲

1. 什么是出口/落地
2. 出口/落地与访问目标之间的关系
3. CNIX 适合的典型场景
4. Po0 国际出口适合的典型场景
5. RFC 香港、日本等方向的选择思路
6. 如何结合入口地区一起判断
