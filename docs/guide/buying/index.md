# 选购指南

这一组文章帮你在下单前判断：Po0 适不适合你、该选哪个入口地区、出口或落地怎么搭配，以及预算应该花在哪里。

## 推荐阅读顺序

1. [Po0 适合谁](/guide/buying/who-is-po0-for)
2. [入口地区怎么选](/guide/buying/choose-entry-region)
3. [出口与落地怎么选](/guide/buying/choose-exit-location)
4. [成本、流量与稳定性](/guide/buying/cost-vs-stability)
5. [用状态页与 Looking Glass 判断效果](/guide/buying/status-and-looking-glass)

## 先记住一句话

Po0 的重点不是“买一台机器”，而是买一条更可控的入口链路。入口地区、出口方向、流量档位和使用限制都要一起看。

如果你还不确定怎么选，先准备：主要使用地区、目标出口地区、预计月流量，再去看具体文章。
