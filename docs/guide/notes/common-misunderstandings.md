# 常见误区

## 用途

解释 Po0 使用中常见的认知偏差，帮助用户建立合理预期，减少因为单次测试或片面指标造成的误判。

## 适合人群/场景

- 第一次购买网络服务，希望理解体验边界的用户。
- 看到延迟、丢包或路径变化后不确定是否异常的用户。
- 需要向团队成员解释服务预期的客户。

## 文章大纲

1. 误区一：延迟越低就一定越好
2. 误区二：一次测试结果可以代表长期体验
3. 误区三：所有目标站点都应该表现一致
4. 误区四：本地网络问题一定能通过服务侧解决
5. 误区五：价格越高就一定适合所有场景
6. 如何用更合理的方式评估体验
