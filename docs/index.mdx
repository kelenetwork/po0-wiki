## 当前定位

- **知识库**：记录项目说明、常见问题、操作流程与排障经验。
- **运维入口**：后续可接入 TCPing、HTTP 探测、延迟图表与告警摘要。
- **静态优先**：默认构建为静态文件，部署简单、可缓存、易迁移。
